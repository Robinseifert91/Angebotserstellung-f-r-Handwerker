# Bild-Analyse Skill

Bei Fotos von Räumen oder Objekten immer ermitteln:
- Geschätzte Fläche (Wände, Boden, Decke) aus Proportionen
- Sichtbarer Zustand: Risse, alte Schichten, Feuchtigkeitsschäden
- Erkennbare Materialien: Fliesen, Putz, Holz, Beton
- Besonderheiten: Steckdosen, Heizkörper, Türen, Fenster (Abzugsflächen)

Ausgabe immer strukturiert:
- Gewerk: [erkanntes Gewerk]
- Geschätzte Hauptfläche: [X m²]
- Auffälligkeiten: [Liste]
- Empfohlene Positionen: [Liste]
- Unsicherheiten: [was nicht erkennbar ist]

Bei Unsicherheit: explizit angeben, nicht schätzen ohne Hinweis.
