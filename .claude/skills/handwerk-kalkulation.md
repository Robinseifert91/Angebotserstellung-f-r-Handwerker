# Handwerk Kalkulation Skill

Pflichtangaben auf deutschen Angeboten:
- Vollständiger Firmenname, Adresse, Steuernummer
- Angebotsnummer, Datum, Gültigkeitsdauer
- Leistungsbeschreibung je Position
- Menge, Einheit, Einzelpreis, Gesamtpreis
- MwSt.-Ausweis (19% oder 7%)
- Zahlungsbedingungen

Kalkulationslogik:
- Materialkosten + Lohnkosten + Aufschlag (15–25%) = Positionspreis
- Arbeitszeit in Stunden, Stundensatz je Gewerk hinterlegen
- Immer Netto + MwSt. getrennt ausweisen

Typische Positionen je Gewerk:
- Maler: Untergrundvorbereitung, Grundierung, Anstrich (Schichten), Abdeckarbeiten
- Fliesenleger: Untergrundvorbereitung, Abdichtung, Fliesenverlegung, Verfugung, Silikon
- Elektriker: Leitungsverlegung, Unterputz/Aufputz, Anschluss, Prüfung
- SHK: Demontage, Montage, Anschluss, Druckprüfung, Inbetriebnahme
